# 📘 Cartilha Informativa sobre REURB

Este repositório foi criado para armazenar o desenvolvimento de um **site informativo** com o objetivo de apresentar uma **cartilha digital sobre a REURB (Regularização Fundiária Urbana)**.

## 📌 Objetivo do Projeto

O projeto tem como finalidade **divulgar informações de forma clara e acessível sobre a REURB**, ajudando a população a compreender:

- O que é a REURB  
- Como funciona o processo de regularização fundiária  
- Quem pode participar  
- Quais são os benefícios da regularização de imóveis  

A proposta é transformar essas informações em uma **cartilha online**, organizada em formato de site.

## 🧠 Motivação

Muitas pessoas vivem em imóveis que ainda **não possuem regularização formal** e desconhecem os procedimentos e direitos envolvidos.  
Este projeto busca **simplificar o entendimento sobre o tema**, utilizando linguagem clara e estrutura visual acessível.

## 🛠️ Tecnologias Utilizadas

O projeto poderá utiliza tecnologias como:

- HTML  
- CSS  
- JavaScript
